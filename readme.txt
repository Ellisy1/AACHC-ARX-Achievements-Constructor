Скрипт для конструирования системы достижений Аркса. 

AACHC.py - рабочий скрипт py 
readme.txt - документация
AACHC_in.txt - какие достижения должны быть? См. ниже

# === # КАК ИСПОЛЬЗОВАТЬ # === #
    В AACHC_in ввести в одну строку:
        ach_айди (придумать самому) 
        name=имя (отображается как название достижения)
        descr=описнание (очевидно, описание достижения)
        xp=42 (получаемый опыт за выполнение конкретного достижения)
        Один из вариантов ниже:
            trigger=tag=тег (проверка, есть ли у игрока конкретный тег)
            trigger=scores=переменная=значение (проверка, равна ли переменая игрока указанному значению. Подддерживает диапазоны, например 20..100)
            trigger=item=arx:предмет (проверка, есть ли указанный предмет у игрока в инвентаре)
            trigger=external (не устанавливаем никакого триггера. Подразумевается, что триггер активируется каким-то внешним способом)
